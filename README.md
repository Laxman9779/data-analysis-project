This project performs data cleaning and exploratory data analysis (EDA) on the Adult Income dataset.

1. Data Cleaning:
- Handled missing values using mode (categorical) and median (numerical)
- Removed duplicate records
- Converted incorrect data types
- Replaced '?' values with NaN

2. Exploratory Data Analysis:
- Performed skewness analysis to understand distribution
- Identified outliers using IQR method
- Created visualizations: line plot, bar chart, histogram, scatter plot, and box plot

3. Correlation Analysis:
- Generated correlation heatmap for numerical features
- Observed relationships between variables

4. Final Output:
- Clean dataset saved as cleaned_data.csv for further modeling
